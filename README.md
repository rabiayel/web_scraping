# 🕸️ Web Scraping with Python

Bu repo, Python kullanarak web scraping (web kazıma) tekniklerini içermektedir.

## 📁 İçerik

- `Webscraping.ipynb` — Web scraping temel uygulamaları
- `EdayaVeda.ipynb` — Ek scraping örneği
- `server.ipynb` — Sunucu taraflı uygulamalar
- `index.html`, `indexx.html` — Örnek HTML sayfaları
- `cars.xls` — Kazınan araç verisi

## 🛠️ Kullanılan Kütüphaneler

- BeautifulSoup
- Requests
- Pandas

## 🚀 Nasıl Çalıştırılır?

Jupyter Notebook ile `Webscraping.ipynb` dosyasını açın.
